# ga4_web

### 環境構築
```
#Vue CLIのインストール(vueが入っていなければ)
$ npm install -g @vue/cli 5.0.4

#パッケージのインストール
$ npm install
```

### ローカルサーバーの立ち上げ
```
npm run serve
```
